九龙湖校区

发表于 2024-07-07 分类于 随感

夏天正午，阳光炙烤着大地，校园平坦而又空旷，像一个铁皮饼干盖，太阳可以地照遍它的每个角落，而不留一处阴凉，这个世界是明晃晃的，让人头晕。

这里的一切都是那么直白，瞟上一眼就已经完全了解，光天化日之下没有什么秘密。

学校的中央有一块巨大而又平坦的草坪，纯粹的草坪，很容易误以为是足球场，如果人能长到和三楼一样高的话，俯瞰这片草坪或许还算好看。大部分时间草坪上都很空旷，只有在晚上的时候才会有学生散坐在这。

校园的四面挖了一圈河沟，这个护校河把学校变成了一座孤岛，周围都是方块形状的工厂，走在马路上，迎面过来的只有各式各样的卡车，留下的只有一串连一串的尾气。

我常常在想，一直在这里的学习生活的人知不知道河的外面还有另外一个世界。

世界的线

发表于 2024-07-07 分类于 随感

所有的世界如同一个巨大的球体，而每个人所经历的则是球体上行进的一条线。我们从同一位置出发，在球体上游走，有时向前，有时后退，有时徘徊，有时停滞，最终在球体的另一端相遇，构成了自己独特的世界之线。

有些人的线条直来直去，而有些人的线条蜿蜒曲折。然而，相对于整个球体而言，每个人走过的线几乎为零。无论你见过多少世界，实际上也只是所有世界中微不足道的一点。

但若努力让自己的线变得蜿蜒曲折，至少在看到另一条轨迹时，不至于惊讶地叫出声，或是遗憾地叹息。

晚饭后

发表于 2024-07-07 分类于 随感

夏天的晚上，虽然快八点了，太阳还是迟迟不愿离开，还在远处斜斜的晒着，但已经没有什么热量了，我想出门吃完面。

在面馆点了一碗素椒杂酱面，几口就吃完了，吃饱了慢慢走回去。

夜幕逐渐降临，天边的晚霞慢慢消退，走回校园，夜色中路灯发出昏黄的光，树叶投下斑驳的光影，微微摇晃，蝉鸣声此起彼伏，夜晚的视线变得模糊，周围的世界也变得暧昧起来。

走到体育馆附近，孩子们嬉戏，情侣们依偎，老人们纳凉，所有人共同分享同一片草地，都在享受着夜晚，就算是一个理想主义的坚定怀疑者，看到这一幕，恐怕也会有些许动摇。

成都的冬天

发表于 2024-04-19 分类于 随感

成都的冬天并不美丽，很少出太阳，抬头看天空，总是白茫茫一片，很难分辨现在是上午还是下午。

成都的天黑得晚，夏天的晚上，太阳一直不肯离去，七八点钟还在斜斜的晒着，把人都要烤晕。而在没有太阳的冬天，只是微微的凉风，一件棉袄就足够，温吞地，不紧不慢地，茫茫的一天又要过去，这常常让人感到忧郁和迷失，但这种感觉会在晚上，一盆红彤彤的冒菜端上来的一瞬间消失。
