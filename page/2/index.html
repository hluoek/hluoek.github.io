一些边缘

发表于 2024-04-18 更新于 2024-04-19 分类于 随感

城市的边缘，一辆面包车在红灯前停下，几个年轻人从车里钻出，背着装着衣服和被子的大包裹，他们环顾四周：绿色围挡围住了正在施工的马路，只留下了一条狭窄的步行通道，几座吊车指向灰蒙蒙的天空，上面挂着不存在的旗帜，未完工的高架桥底座孤零零的矗立着，像一座沉默的墓碑。

然后他们向这里走来，他们来自安徽河南的农村，他们去往工厂和车间，去往现代化的荒原。

整片区域排布着上千栋楼，统一规格的楼房默默等待着不同的人，布告栏和转角处贴着花花绿绿的招工和招租广告，只需要20元就可以度过一个寒冷的夜晚。

并排的几条巷子是他们暂时的落脚点，河南烩面，川湘菜馆和沙县小吃相邻排布，里面吃饭的人总是不少。

一排名叫某某人力资源的商铺，门口张贴着各种工厂的招聘广告，日结月结，长白班夜班，单休无休，电子厂操作工，工地小工，服装厂配货工，装修瓦工…

突然伸出一个不知通向哪里的楼梯，二楼的角落里，地上铺着被子和枕头，几个人坐在床铺上吃面。

有一家叫交换商店的店铺，上面横幅写着“交换故事，交换快乐”，在这里可以将身上的物品换成钱。

这是南京的“三和”，刚来到这座城市的人们在这里暂时落脚，然后被分配到附近的工厂。这里人员密集，建设杂乱，路过的人也并不愿意多驻足停留。

一些比较边缘的青春在这里发生着，去寻找，去等待。现代化的口号每晚准时地从电视机中响起，我却看到一双双黯淡的眼睛和粗糙的手。我突然感到一阵疼痛，来自不会说话的伤口。

链接预测任务的数据集划分

发表于 2024-04-17 更新于 2024-04-18 分类于 技术

  • train的message edge和supervision_pos_edge可以相同也可以不同，在SEAL中二者不同，在GAE中二者相同。

  • train，val，test的supervision/evaluate必须不同，否则会数据泄露。

  • 一般默认的数据集分割方法

    • val_message=train_message + train_supervision_pos_edge 如果后两者相同，则和三者均相等

    • test_message=train_message+train_supervision_pos_edge+val_evaluate_pos_edge

  • 数据集划分中的比例是指各部分supervision/evaluate的比例

  • 是否具有train_neg_edge?
